01 — 時態總覽
Présent 現在式 現在習慣、事實、此刻動作
Passé récent 最近過去式 venir de + inf ／ 剛發生的事
Passé composé 複合過去式 avoir/être + p.p. ／ 過去已完成動作
Imparfait 未完成過去式 過去持續 / 習慣 / 背景描述
Plus-que-parfait 愈過去式 avoir/être（未完成）+ p.p. ／ 過去的過去
Futur proche 近未來式 aller + inf ／ 即將發生
Futur simple 簡單未來式 原形 + 未來語尾 ／ 未來將發生
Conditionnel présent 條件式現在 假設語氣 / 禮貌請求
Conditionnel passé 條件式過去 avoir/être（條件式）+ p.p. ／ 未實現假設
Subjonctif présent 虛擬式現在 vouloir que / il faut que 後的子句
02 — 時態句型結構

Présent 現在式

肯定
S + [代名詞] + V
Je le mange.
否定
S + ne + [代名詞] + V + pas
Je ne le mange pas.

Passé composé 複合過去式

肯定
S + [代名詞] + avoir/être + p.p.
Je l'ai mangé. / Elle est partie.
否定
S + ne + [代名詞] + avoir/être + pas + p.p.
Je ne l'ai pas mangé.

Imparfait 未完成過去式

肯定
S + [代名詞] + V-ais/ait…
Je le mangeais.
否定
S + ne + [代名詞] + V + pas
Je ne le mangeais pas.

Futur proche 近未來式 ⚠️

肯定
S + [代名詞] + aller + inf
Je vais le manger.
否定 代名詞位置改變！
S + ne + aller + pas + [代名詞] + inf
Je ne vais pas le manger.

Futur simple 簡單未來式

肯定
S + [代名詞] + V-rai/ras…
Je le mangerai.
否定
S + ne + [代名詞] + V + pas
Je ne le mangerai pas.

Conditionnel présent 條件式現在

肯定
S + [代名詞] + V-rais/rait…
Je le mangerais. / Je voudrais un café.
否定
S + ne + [代名詞] + V + pas
Je ne le mangerais pas.

Passé récent 最近過去式 ⚠️

肯定
S + [代名詞] + venir de + inf
Je viens de le manger.
否定 代名詞位置改變！
S + ne + venir + pas de + [代名詞] + inf
Je ne viens pas de le manger.
